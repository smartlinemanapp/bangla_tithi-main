# Bangla Tithi Calendar

The most accurate and beautiful Bangla Tithi Calendar application.

## Features
- **Accurate Tithi**: Calculations based on standard algorithm.
- **Solar & Lunar**: Detailed sunrise, sunset, and moon phases.
- **Offline Ready**: Works perfectly without internet.
- **Dark Mode**: Beautiful themed interfaces.
